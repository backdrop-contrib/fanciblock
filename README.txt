DESCRIPTION
-----------

Fanciblock module can be used to create different Content box with Icons

You can try out live demonstration of it here:
http://vrinda.backdropchamp.com/content-boxes


INSTALLATION
------------

1. Download & extract "Font-Awesome" from http://fortawesome.github.com/Font-Awesome/
and place inside "sites/all/libraries/fontawesome" directory. CSS file should be
sites/all/libraries/fontawesome/css/font-awesome.css

2. Enable the module at Administer >> Site building >> Modules.
   (http://backdrop.org/documentation/install/modules-themes/modules-7)

3. For Settings: admin/config/user-interface/fanciblock/


That's it!
